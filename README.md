# Number Guessing Game (Python)

## Description
This is a simple console-based Number Guessing Game developed using Python.
The program generates a random number, and the user has to guess it within
a limited number of attempts.

## Features
- Random number generation
- User input validation
- Limited number of attempts
- Helpful hints (Too high / Too low)

## How to Run
1. Make sure Python is installed on your system
2. Download or clone this repository
3. Run the program using:
   python number_guessing_game.py

## Technologies Used
- Python
